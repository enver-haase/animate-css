# Animate-CSS

Vaadin 25 Java integration of
https://animate.style/

## Development instructions

Requires Java 21+.

#### Installing the add-on:
Go to the `animate-css` subfolder and do
```
mvn clean install
```

#### Starting the demo:
Go to the `animate-css-demo` subfolder and do

```
mvn spring-boot:run
```

Find the demo at http://localhost:8080 .
